# 命令行工具
<!-- 角色：使用配置者 | 最近验证：2025-12-11 -->

本文档介绍 warp-flow 系统中各种命令行工具的使用方法。

## 工具概览

- wparse - 数据解析引擎（daemon/batch）
- wpgen - 数据生成器（rule/sample/conf/data）
- wproj - 工具包与运维（rule-verify/parse-run/sinks/sources/data/stat/validate/conf/knowdb）
- wprescue - 救援/补写工具（batch）
